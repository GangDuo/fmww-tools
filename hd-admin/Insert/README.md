# CSVを読み込んでテーブルにinsertする

## 準備
実行ファイルと同じディレクトリに、データベース接続情報を配置する


```xml:db.config
<?xml version="1.0" encoding="utf-8"?>
<appSettings>
    <add key="db:server" value= ""/>
    <add key="db:port" value=""/>
    <add key="db:user" value= ""/>
    <add key="db:password" value= ""/>
    <add key="db:database" value= ""/>
</appSettings>
```
