ファッションマネージャーのCSVアップロード系操作をCLIで使用する。



| 短いオプション | 長いオプション   | 説明                     |
| -------------- | ---------------- | ------------------------ |
| -i             | --inventory      | 棚卸インポート           |
| -p             | --pop            | Cポップインポート        |
| -s=VALUE       | --source=VALUE   | データソースファイルパス |

使用例

```powershell
Uplink.exe -i -s=C:\source.csv
```
